Flex With Images

The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

Here we have a div with "flex-container" class which sets the display property to "flex". Here are 5 images of nature in a flex container. The images are equally spaced by setting the justify-content property to "space-around".
